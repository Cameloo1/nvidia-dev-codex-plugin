# Source Policy

Recommendations must be grounded in inspectable sources.

Priority order:

1. Local SDK docs, headers, samples, and repo files.
2. Official NVIDIA developer pages, documentation, release notes, and GitHub repositories.
3. Official engine documentation and NVIDIA engine integration pages.
4. NVIDIA Developer Forums as dated advisory troubleshooting context only.

Every major plan should include sources, assumptions, missing information, and compatibility state.

If local headers disagree with generic docs, prefer the local version-specific headers for code-level guidance and report the disagreement.
